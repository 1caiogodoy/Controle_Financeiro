# 💰 Personal Finance Analyzer

## 📌 Descrição

O Personal Finance Analyzer é uma aplicação web simples desenvolvida em Python utilizando Flask, com o objetivo de ajudar usuários a visualizar e entender melhor sua organização financeira.

A aplicação permite inserir dados como renda, gastos, investimentos, lazer e dívidas, gerando gráficos que facilitam a análise da situação financeira atual e sugerem uma distribuição mais equilibrada.

---

## 🎯 Problema

Muitas pessoas possuem dificuldade em controlar seus gastos e entender para onde seu dinheiro está indo, o que pode resultar em endividamento e falta de planejamento financeiro.

---

## 💡 Solução

A aplicação permite:

- Visualizar a distribuição dos gastos
- Identificar desequilíbrios financeiros
- Obter uma sugestão de reorganização financeira
- Receber alerta caso os gastos ultrapassem a renda

## 🚀 Como executar o projeto

### 1. Clonar o repositório

git clone https://github.com/1caiogodoy/Controle_Financeiro.git

cd Controle_Financeiro

### 2. Criar ambiente virtual

python -m venv venv

### 3. Ativar o ambiente

Windows:

venv\Scripts\activate

Linux/Mac:

source venv/bin/activate

### 4. Instalar dependências

pip install -r requirements.txt

### 5. Executar aplicação

python main.py

Acesse no navegador:

http://127.0.0.1:5000/
